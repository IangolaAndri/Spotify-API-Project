# Spotify-API-Project
Python project using Spotify API OAuth authentification to retrieve an user's Spotifify data:
* Top artists from last 4 months/6monts/all time
* Top tracks from last 4 months/6monts/all time
* Public playlists

## How to run 
% Python3 Routes.py
